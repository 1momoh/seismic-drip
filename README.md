# 💧 Seismic Quick Faucet

A lightweight testnet ETH faucet for the [Seismic testnet](https://docs.seismic.systems/appendix/testnet). Users can claim **0.001 ETH** every 24 hours directly to their wallet.

---

## How It Works

### Claiming
1. Enter a valid EVM wallet address (`0x...`)
2. Solve the simple math captcha
3. Click **Claim 0.001 ETH**

The faucet enforces a **24-hour cooldown per IP address and per wallet address**, so the same address or network cannot drain the faucet repeatedly.

On a successful claim, a link to the transaction on the [Seismic explorer](https://seismic-testnet.socialscan.io) is shown.

### Donating
If you have spare testnet ETH and want to keep the faucet running, click **Copy Donate Address** on the donate card and send ETH directly to the faucet contract.

The faucet balance is displayed in real time and refreshes every 30 seconds.
